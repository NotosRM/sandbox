Создай два эксперимента для сравнения двух технологий на одинаковом сценарии.

Параметры: $ARGUMENTS
Ожидаемый формат: `<tech1> vs <tech2> [шаблон] [сценарий]`
Пример: `zustand vs reatom react-vite counter with increment/decrement`
Если шаблон не указан — используй `react-vite`.

Шаги:

1. Разбери аргументы: определи tech1, tech2, шаблон, сценарий.
2. Создай эксперимент `<tech1>-vs-<tech2>-a` из указанного шаблона:
   `pnpm run create -- --template <шаблон> --name <tech1>-vs-<tech2>-a`
3. Создай эксперимент `<tech1>-vs-<tech2>-b` из того же шаблона:
   `pnpm run create -- --template <шаблон> --name <tech1>-vs-<tech2>-b`
4. Выполни `pnpm install` из корня.
5. Установи <tech1> в первый: `cd experiments/<tech1>-vs-<tech2>-a && pnpm add <tech1>`
6. Установи <tech2> во второй: `cd experiments/<tech1>-vs-<tech2>-b && pnpm add <tech2>`
7. Реализуй одинаковый сценарий в обоих экспериментах.
8. Напиши одинаковые тесты в обоих, проверяющие поведение сценария (используй @testing-library/react для react-vite).
9. Запусти тесты обоих: `pnpm test:run` из каждого.
10. Обнови README обоих: заполни "Цель" (что сравниваем) и "Стек" (какая библиотека).
11. Выведи: как запустить оба, как прогнать тесты, на что обратить внимание при сравнении.
