Сделай сводку по всем экспериментам в песочнице.

Параметры: нет

Шаги:

1. Прочитай `experiments/*/README.md` и `experiments/*/package.json` для всех экспериментов.
2. Для каждого эксперимента определи:
   - Имя (из имени директории)
   - Шаблон (из `package.json` поля `sandbox.template`)
   - Статус (из README: in-progress / done / archived)
   - Цель (краткая, из README секции "Цель")
   - Наличие тестов (есть ли `*.test.ts(x)` файлы в `src/`)
   - Наличие E2E (есть ли папка `e2e/`)
3. Сгруппируй по статусу: in-progress, done, archived.
4. Выведи таблицу:

| Имя | Шаблон | Статус | Цель | Тесты | E2E |
| --- | ------ | ------ | ---- | ----- | --- |
| ... | ...    | ...    | ...  | ✓/—   | ✓/— |

5. Если у экспериментов со статусом `done` не заполнена секция "Выводы" — укажи их отдельным списком.
